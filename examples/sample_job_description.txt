NovaCraft Technologies is a fast-growing Series C startup building the next generation of cloud-native collaboration tools for enterprise teams. About the Role

We are looking for a Senior Software Engineer to join our Platform team. You will design, build, and maintain the core services that power our real-time collaboration engine. This is a high-impact role where you will own significant portions of the architecture and mentor junior engineers on the team.

Responsibilities

- Design and implement scalable microservices using TypeScript and Node.js on AWS infrastructure
- Build and optimize high-performance RESTful and GraphQL APIs serving millions of requests per day
- Lead the development of real-time features using WebSockets and event-driven architecture
- Architect and manage PostgreSQL databases, including schema design, query optimization, and migration strategies
- Drive adoption of infrastructure-as-code practices using Terraform and AWS CDK
- Collaborate with product managers and designers to translate requirements into technical specifications
- Mentor junior engineers through code reviews, pair programming, and technical design sessions
- Champion engineering best practices including CI/CD, automated testing, and observability
